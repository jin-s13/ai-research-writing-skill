# AI Research Writing Skill Wrapper

This wrapper exists for plugin hosts that discover skills from a `skills/` directory.

Load and follow the repository root `../../SKILL.md`. That root file is the canonical skill contract for operating modes, gates, evidence policy, reference loading, task management, scripts, and output discipline.

Use `../../references/README.md` as the reference map and `../../scripts/README.md` for helper scripts.
