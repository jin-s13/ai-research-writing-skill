# Template Sources and License Notes

This file records the intended source and verification status for bundled LaTeX templates. Conference templates change over time; always check the official venue page before active submission.

## Verification Policy

- Prefer official venue repositories or author-kit downloads.
- Preserve upstream license files when they are provided.
- Do not assume page limits, checklist requirements, or anonymity options are current only because a local template compiles.
- If a template source cannot be verified, treat the bundled copy as a convenience draft only.
