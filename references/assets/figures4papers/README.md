# Bundled Figure Pattern Assets

This directory vendors figure-pattern scripts used as local references for publication-style deterministic result figures.

Primary provenance:

- Vendored from `Yuan1z0825/nature-skills`, `skills/nature-figure/assets/figures4papers/`
- Upstream pattern source: `ChenLiu-1996/figures4papers`

Use these files as pattern references for layout, palette, axes, legend handling, and export conventions. Do not reuse demo data, manuscript-specific labels, or paper-specific claims in a target manuscript.

When adapting a pattern:

1. Start from the target paper's figure contract and core conclusion.
2. Choose the closest chart family or layout pattern.
3. Rewrite the script around the target paper's own data and terminology.
4. Keep source data, plotting script, and exported assets traceable.

Acknowledgement should be preserved in this repository's README files.
