# Minimal Paper Repo Demo

This is a tiny, safe demo project for AI Research Writing Skill.

It mimics a small ML research repository with:

- a toy method description;
- a deterministic result table;
- a draft LaTeX entry point;
- expected output artifacts generated by the skill.

Try this prompt from the repository root:

```text
Use AI Research Writing Skill to inspect examples/minimal-paper-repo and produce or update paper_story.md, claim_evidence_map.md, literature/positioning.md, figures/figure_plan.md, and build_check.md.
```

The numbers are illustrative demo values, not scientific claims.
