# Example: Writing a Paper About This Skill

This example answers the question:

> If we used AI Research Writing Skill to write a paper about `ai-research-writing-skill` itself, what would the output look like?

It is a complete meta-paper package for the current repository. The paper frames the project as a system/workflow contribution: a claim-evidence-engineering skill that helps agents turn ML/AI research repositories into auditable paper artifacts.

## Prompt

```text
Use AI Research Writing Skill to write a complete system paper about this repository itself.
Treat ai-research-writing-skill as the research artifact.
Inspect SKILL.md, references/, scripts/, templates/, cross-platform plugin files, and README.
Create paper_story.md, claim_evidence_map.md, literature positioning, verified citations, ICML-style figures/tables, and a build-ready ICML LaTeX paper under examples/paper-about-ai-research-writing-skill/paper/.
Do not invent performance numbers. Use repository facts as evidence.
```

## Final Paper Package

- `paper/main.tex`: complete ICML-style LaTeX paper draft.
- `paper/references.bib`: citations for related open-source projects and official metadata.
- `paper/figures/teaser_imagegen.png`: generated teaser figure.
- `paper/figures/overview_imagegen.png`: generated method overview figure.
- `paper/figures/method_overview.tex`: editable TikZ fallback for the method overview.
- `paper/figures/artifact_contract.tex`: durable artifact contract figure.
- `paper/tables/repository_evidence.tex`: repository evidence table.
- `paper/tables/related_projects.tex`: related-project positioning table.
- `paper/main.pdf`: generated by compiling the paper.

## Audit Artifacts

- `paper_story.md`: thesis, gap, contribution, and claim boundaries.
- `claim_evidence_map.md`: every major claim tied to repository evidence.
- `literature/positioning.md`: how this project differs from related projects.
- `citation_verification.md`: citation source and verification status.
- `evidence/repository_inventory.md`: local files used as paper evidence.
- `build_check.md`: compile command and result.

This is an example artifact, not a submitted paper.
